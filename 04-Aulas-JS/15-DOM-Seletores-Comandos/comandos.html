Anotações de alguns comandos para HTML, CSS E JS

No mundo do DOM (Manipulação de HTML)

innerText,Pega ou define apenas o texto de um elemento.,"paragrafo.innerText = ""Olá!"""

innerHTML,Pega ou define o HTML interno (reconhece tags).,"div.innerHTML = ""Negrito"""

style,Altera o CSS diretamente via JS.,"box.style.color = ""red"""

classList,"Adiciona, remove ou alterna classes CSS.","el.classList.add(""ativo"")"

value,Pega o que o usuário digitou em um input.,inputNome.value


No mundo dos Arrays (Listas)

push(),Adiciona um item ao final do array.,"lista.push(""Novo"")"

join(),Junta os itens do array em uma única string.,"[""A"", ""B""].join(""-"") -> ""A-B"""


includes, O .includes() serve para verificar se algo está contido dentro de outra coisa. Ele sempre devolve um valor Booleano (true ou false). Ele verifica se um "pedaço" de texto existe dentro de outro, não importa a posição.

No mundo das Strings (Textos)

toLowerCase(),Transforma tudo em minúsculo.,"""OI"".toLowerCase() -> ""oi"""

toUpperCase(),Transforma tudo em MAIÚSCULO.,"""oi"".toUpperCase() -> ""OI"""

trim(),Remove espaços vazios inúteis no início e fim.,""" oi "".trim() -> ""oi"""

includes(),Verifica se um texto existe dentro de outro.,"""Banana"".includes(""nan"") -> true"

split(),Quebra um texto e transforma em um Array.,"""A,B,C"".split("","") -> [""A"", ""B"", ""C""]"
